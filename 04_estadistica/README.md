# Nivel 4: Estadística y Probabilidad

Este módulo conecta la teoría estadística con su implementación práctica en Python, utilizando `scipy.stats` y `statsmodels`.

## Contenido

1.  **[Estadística Descriptiva y Distribuciones](./04_01_estadistica_descriptiva.ipynb)**
    *   Medidas de tendencia central y dispersión.
    *   Visualización de distribuciones (histogramas, boxplots).
    *   Distribuciones de probabilidad clave (Normal, Binomial, Poisson) y sus propiedades.

2.  **[Inferencia Estadística y Test de Hipótesis](./04_02_inferencia_test_hipotesis.ipynb)**
    *   Teorema Central del Límite.
    *   Intervalos de confianza.
    *   Tests de hipótesis (T-test, Chi-cuadrado).
    *   p-valores y significancia estadística.

## Objetivos de Aprendizaje
Aprenderás a describir datos numéricamente, entender el comportamiento aleatorio mediante distribuciones y tomar decisiones basadas en datos utilizando pruebas estadísticas rigurosas.
