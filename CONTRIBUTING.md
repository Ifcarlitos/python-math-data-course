# Contribuir al Curso

¡Gracias por tu interés en contribuir! Para mantener la calidad del curso, por favor sigue estos pasos:

1.  **Fork** el repositorio.
2.  Crea una **rama (branch)** para tu contribución (`git checkout -b mejora-modulo-1`).
3.  Realiza tus cambios y haz **commit**.
4.  Asegúrate de que los notebooks se ejecuten correctamente de principio a fin.
5.  Envía un **Pull Request (PR)**.

## Estilo
*   Usa nombres de archivo descriptivos siguiendo el patrón `MM_NN_nombre.ipynb`.
*   Comenta el código cuando sea necesario.
*   Mantén el tono educativo y riguroso.
